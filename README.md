# Local Setup
![license MIT](https://img.shields.io/badge/license-MIT-lightgrey.svg)

This repository contains [**@rodriados**](https://github.com/rodriados)'s workspace
setup scripts and routines.

## Motivation
When starting to work with a new machine - be it a personal or a work one -, it is
very time demanding to configure it and set up all the softwares and tools I need
and use in a daily basis. It is _specially_ annoying when I cannot remember how to
install them or what versions are the ones I need.

For that reason, I decided to automate the installation and setup routines of all
the softwares, drivers and tools I need in a fresh machine.

## Usage
This project requires `pipx` to be present in your system, then run `setup.sh`.
